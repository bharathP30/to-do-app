## Features

- Add new tasks to the list
- Mark tasks as completed or uncompleted
- Delete tasks from the list
- filter tasks from the list
- Responsive and user-friendly interface
- Data persistence using local storage (tasks saved in browser)

## Technologies Used

- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for dynamic interaction and local storage handling

## How to Use

1. Open `index.html` in a web browser.
2. Enter a new task in the input field and click "Add" or press Enter.
3. Click the checkbox next to a task to mark it complete or incomplete.
4. Use the delete button to remove tasks from the list.
5. Use the filter buttons to filter tasks accordingly
6. Your tasks are saved locally and will persist on page refresh.
